## Part 2: Complete Examples

### Example 1: 교사 - 수업 계획서 작성

**Base Task**: "5학년 국어 3단원 '의견을 조절하며 토의해요' 학습지도안 작성"
**작업 유형**: 문서 작성 (글쓰기)

#### Difficulty 1-2 Variations (자연어)

**GPT-5.2**:
```
당신은 초등학교 5학년 담임교사입니다. 국어 3단원 '의견을 조절하며 토의해요'의 학습지도안을 작성해주세요.
```

**Claude 4.5**:
```
당신은 경력 10년의 초등학교 교사입니다. 5학년 국어 3단원 학습지도안을 작성하세요. 도입-전개-정리 구조로 작성하고, 구체적인 활동을 포함해주세요.
```

**Gemini 3**:
```
당신은 초등학교 5학년 담임교사입니다. 국어 3단원 토의 수업의 학습지도안을 작성하세요. 40분 수업 기준입니다.
```

**Perplexity**:
```
초등학교 5학년 국어 토의 수업에 효과적인 학습지도안 구조를 검색하고 예시를 제공해주세요.
```

**차이점 분석**:
- GPT: 최소 정보, 모델이 추론
- Claude: 명시적 구조 요구 ("도입-전개-정리"), 구체성 강조 ("경력 10년")
- Gemini: 제약 포함 ("40분")
- Perplexity: 검색 + 예시 요청

    ## 학습지도안
    | 단계 | 시간 | 교수-학습 활동 | 자료 및 유의점 |

**차이점 분석**:

### 이 패턴을 다른 직업군에 적용하는 방법 (Job Adaptability)

#### 교사 → 의사 (진료 계획서)
```
"수업 계획서" → "진료 계획서"
"학생 수준" → "환자 상태"
"학습 목표" → "치료 목표"
"40분 수업" → "30분 진료"
"도입-전개-정리" → "문진-검사-처방"
```

#### 교사 → 마케터 (캠페인 기획서)
```
"수업 계획서" → "캠페인 기획서"
"학생" → "타겟 고객"
"학습 목표" → "캠페인 목표"
"40분" → "2주 캠페인 기간"
"학습 활동" → "마케팅 활동"
```

#### 핵심 원칙
**직업이 바뀌어도 구조는 동일합니다:**
1. 역할 (전문성 수준 명시)
2. 시간/기간 제약
3. 목표 (2개 이상)
4. 단계별 활동
5. 산출물 형식

**변경할 요소만 교체하면 됩니다.**

```markdown
# 역할
부동산 전문 변호사 (경력 15년)

## 배경 맥락
부동산 매매는 고액 거래이므로 계약서 검토가 매우 중요합니다. 특히 특약 사항은 표준 계약서에 없는 내용이므로 세밀한 검토가 필요합니다.

## 검토 지시사항
1. 계약서 전체를 읽고 표준 계약서와 다른 점을 식별하세요
2. 각 조항이 민법 및 부동산 관련 법령에 부합하는지 확인하세요
3. 매수인에게 불리한 조항이 있는지 파악하고, 왜 불리한지 설명하세요
4. 수정이 필요한 조항은 구체적인 수정안을 제시하세요

**중요**: 각 지적 사항마다 관련 법령 또는 판례를 근거로 제시하세요. 단순한 의견이 아닌 법적 근거 기반 검토가 필요합니다.

## 출력 형식
### 검토 결과 요약
### 위험 조항 분석 (조항별)
### 수정 권고안
```
