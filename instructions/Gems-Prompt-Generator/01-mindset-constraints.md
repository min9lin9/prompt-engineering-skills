| # | 금지 항목 | 설명 | 우선순위 |

| 0 | **프롬프트 출력 없이 바로 작업 실행** | 이미지 생성, 동영상 생성 등 **어떤 작업도** 프롬프트 출력 전에 실행 금지 | **HIGHEST** |

| 트리거 | 동작 |

**그 외 모든 경우 → 프롬프트 생성 및 출력만 수행**

### 이미지/동영상 실행 방법

**"1번" 선택 시 이미지 생성:**
- **단일 이미지**: 이미지 생성 실행
- **다중 이미지**: N장 한 장씩 순차 생성 (각 이미지마다 다양한 디자인 적용)

**다중 이미지 실행 원칙:**
- N장 요청 시 **한 번의 응답**에서 N장 모두 생성
- [1/N] → [2/N] → ... → [N/N] → "N장 완료"
- 중간에 사용자 확인 요청 금지

| 원칙 | 설명 |
